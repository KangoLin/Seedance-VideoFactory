# 漫剧工具使用说明

本文档用于说明本地漫剧视频生成工具的完整使用流程。工具通过本地网页 GUI 管理集数、分镜任务、参考素材、提示词、图片生成、视频生成、一键拼接和下载。

## 1. 工具用途

本工具主要用于批量制作漫剧分镜视频，适合以下流程：

1. 按集数管理短剧或漫剧项目。
2. 每集拆分为多个分镜任务。
3. 每个分镜上传参考图、参考视频或首尾帧。
4. 编写视频提示词。
5. 调用 Seedance 视频生成 API 生成分镜视频。
6. 使用 Gemini 图片生成辅助产出参考图。
7. 将同一集的多个分镜视频一键拼接为完整成片。
8. 下载单个分镜视频或拼接后的整集视频。

## 2. 启动工具

### 2.1 启动方式

在项目根目录双击：

```bat
start_seedance_gui.bat
```

启动后浏览器访问：

```text
http://127.0.0.1:8765/
```

如果页面没有自动打开，可以手动在浏览器地址栏输入上述地址。

### 2.2 更新后强刷页面

如果工具刚刚更新过功能，浏览器可能还缓存旧页面。请按：

```text
Ctrl + F5
```

强制刷新页面。

### 2.3 服务状态检查

如果怀疑服务没有启动，可以访问：

```text
http://127.0.0.1:8765/api/ping
```

正常情况下会返回包含 `api_version`、`build_id` 和 `features` 的 JSON。

## 3. 需要准备的内容

### 3.1 基础环境

需要提前准备：

- Python 3.10 或以上版本。
- ffmpeg，并确保命令行可用。
- 火山 / Seedance API Key。
- 如需 AI 提示词优化或图片生成，需要 DeepSeek / Gemini API Key。

### 3.2 API Key 文件位置

密钥文件放在项目根目录的 `API_Key/` 下：

```text
API_Key/VE_Key.txt
API_Key/deepseek_api_key.txt
API_Key/gemini_api_key.txt
```

说明：

- `VE_Key.txt`：Seedance 视频生成 API Key。
- `deepseek_api_key.txt`：DeepSeek 提示词优化 Key。
- `gemini_api_key.txt`：Gemini 提示词优化和图片生成 Key。

`API_Key/` 已被 `.gitignore` 忽略，不应提交或外传。

## 4. 页面整体结构

网页工具分为两层：

1. **集数列表页**
   管理第几集、集数标题和集数顺序。

2. **分镜任务页**
   进入某一集后，管理这一集下的所有分镜任务。

每个分镜任务面板通常包含：

- 左侧：视频生成参数、提示词、参考素材、时长、画幅比例、正式生成按钮。
- 右侧：视频预览、下载按钮、图片生成、运行日志。

## 5. 集数管理

### 5.1 新建集数

在首页点击 `+` 添加新集。

弹窗内可以：

- 滚动数字列表选择集数编号。
- 直接输入集数编号。
- 设置显示标题，例如 `第 4 集` 或自定义标题。

### 5.2 修改集数

可以通过以下入口修改集数：

- 集数卡片右上角的编辑按钮。
- 进入集数后顶部的 `修改集数` 按钮。

### 5.3 调整集数顺序

拖动集数卡片左侧的拖拽手柄，可以调整集数显示顺序。

顺序会自动保存到本地。

### 5.4 删除集数

集数卡片右上角有删除按钮。

注意：

- 至少保留 1 集。
- 删除集数会移除该集下的任务配置。
- 已经生成到 `exports/` 的视频文件通常不会被自动删除。

## 6. 分镜任务管理

### 6.1 添加分镜任务

进入某一集后，点击任务列表中的 `+` 添加新的分镜任务。

每个任务会自动编号，例如：

```text
分镜任务1
分镜任务2
分镜任务3
```

### 6.2 删除分镜任务

每个分镜任务标题栏有 `删除` 按钮。

注意：

- 每集至少保留 1 个分镜任务。
- 删除任务会移除该任务配置。
- 已生成的视频文件通常不会被自动删除。

### 6.3 自动保存

以下内容会自动保存到：

```text
05_Video/workspace/episodes.json
```

包括：

- 集数信息。
- 分镜任务信息。
- 视频提示词。
- 视频参考素材。
- 首尾帧。
- 时长。
- 画幅比例。
- 图片生成提示词。
- 图片生成参考图。
- 图片生成结果预览。

页面左上角显示 `已保存到本地` 后，刷新页面不会丢失这些配置。

## 7. 视频生成模式

当前视频生成区支持两种模式：

### 7.1 参考 + 文字

适合使用参考图、参考视频，再配合文字描述生成视频。

支持的参考素材：

| 类型 | 格式 | 数量限制 | 大小限制 |
| --- | --- | --- | --- |
| 参考图 | PNG / JPG / JPEG / WEBP | 最多 9 张 | 单张不超过 30MB |
| 参考视频 | MP4 / MOV / WEBM | 最多 3 个 | 单个不超过 50MB |

参考图和参考视频可以混用。

### 7.2 首尾帧（上传）

适合固定视频开始和结束画面。

需要上传：

- 首帧图片。
- 尾帧图片。
- 视频描述 Prompt。

首尾帧只支持图片，不支持视频。

## 8. 上传参考视频和参考图

### 8.1 文件选择上传

在视频生成区找到：

```text
参考视频和参考图
```

点击文件选择按钮，可以选择：

- 图片文件。
- 视频文件。
- 多个文件同时选择。

当前文件选择器默认支持：

```text
image/*, video/*, .png, .jpg, .jpeg, .webp, .mp4, .mov, .webm
```

### 8.2 剪贴板粘贴图片

如果剪贴板里是图片，可以：

1. 鼠标移入左侧视频生成区。
2. 按 `Ctrl + V`。
3. 图片会追加到视频参考素材列表。

注意：

- 剪贴板粘贴目前只支持图片。
- 视频请通过文件选择上传。

### 8.3 删除参考素材

每个参考素材缩略图右上角有删除按钮。

点击后会从当前任务中移除该素材。

## 9. 视频提示词 Prompt 编写

每个分镜任务都有：

```text
视频描述（Prompt）
```

建议写清楚：

- 时间段，例如 `0-2s`、`2-4s`。
- 镜头类型，例如特写、中景、低角度、过肩镜头。
- 角色动作。
- 表情变化。
- 场景变化。
- 风格，例如低多边形 3D、卡通渲染、电影感。
- 是否严格参考素材。

示例：

```text
0-2s，中景镜头，主角从走廊尽头跑来，神情紧张。
2-4s，切到特写，主角回头看向身后，眼神恐惧。
低多边形3D渲染，人物形象和场景严格参考参考素材。
```

### 9.1 严格参考素材按钮

点击：

```text
+ 严格参考素材
```

会自动在 Prompt 后追加类似约束：

```text
人物形象和场景严格参考参考素材，杜绝美化和修改
```

## 10. 时长设置

每个分镜任务可设置：

```text
时长（秒）
```

支持范围：

```text
4 - 15 秒
```

常用建议：

- 快节奏分镜：4-6 秒。
- 信息量较多的动作：6-9 秒。
- 复杂长镜头：10-15 秒。

## 11. 画幅比例设置

每个分镜任务可单独选择：

```text
画幅比例
```

可选项：

| 比例 | 适合场景 |
| --- | --- |
| 9:16 | 抖音、TikTok、短视频竖屏 |
| 16:9 | 横屏视频 |
| 1:1 | 方形内容 |
| 3:4 | 偏竖构图 |
| 4:3 | 传统横向画面 |

默认是：

```text
9:16
```

## 12. 干跑检查

点击：

```text
干跑检查
```

会进行参数和素材校验，但不会真正提交 API 生成视频。

适合用于检查：

- Prompt 是否为空。
- 参考素材是否缺失。
- 首尾帧是否缺失。
- 素材类型是否支持。
- 请求 payload 是否正确。

## 13. 正式生成

点击：

```text
正式生成
```

工具会：

1. 检查当前任务参数。
2. 编码参考图片。
3. 处理参考素材。
4. 提交 Seedance API。
5. 轮询任务状态。
6. 下载生成结果。
7. 导出到 `05_Video/exports/`。
8. 在任务右侧显示视频预览。

### 13.1 运行日志

每个任务右侧下方都有运行日志。

日志会显示：

- API 是否已提交。
- 正在编码第几个参考素材。
- 是否正在处理参考素材。
- Seedance 任务状态。
- 成功或失败原因。

## 14. 下载单个分镜视频

分镜生成成功后，视频预览下方会出现：

```text
下载当前分镜视频
```

点击后，浏览器会保存到默认下载目录。

注意：

- 下载位置由浏览器设置决定。
- 如果浏览器设置了“下载前询问保存位置”，会弹出保存对话框。
- 如果没有开启询问，通常会保存到系统 `Downloads / 下载` 文件夹。

## 15. 一键拼接本集

进入某一集后，顶部有：

```text
一键拼接本集
```

点击后，工具会按分镜顺序拼接当前集的最新预览视频：

```text
分镜任务1 → 分镜任务2 → 分镜任务3 → ...
```

输出文件位于：

```text
05_Video/exports/
```

文件名示例：

```text
ep-4-concat_TikTok_v001.mp4
```

多次拼接会自动递增版本号：

```text
v001
v002
v003
```

### 15.1 下载拼接成片

拼接成功后，结果区会显示：

- 新窗口播放。
- 下载视频。
- 视频预览。

点击 `下载视频`，会保存到浏览器默认下载目录。

## 16. AI 提示词优化

每个任务面板都有：

```text
AI 提示词优化
```

可选择：

- DeepSeek。
- Gemini。
- 对应模型。

用途：

- 分析当前 Prompt 问题。
- 优化镜头语言。
- 强化角色一致性。
- 改善节奏、动作、光影和构图。

使用流程：

1. 填写视频 Prompt。
2. 点击 `AI 提示词优化`。
3. 选择模型。
4. 点击发送。
5. 查看返回结果。
6. 点击应用优化提示词。

## 17. 图片生成（Gemini）

每个分镜任务右侧有：

```text
图片生成（Gemini）
```

当前模式固定为：

```text
参考图 + 文本
```

也就是说，图片生成需要：

- 上传或粘贴图片参考。
- 输入图片提示词。

### 17.1 添加图片生成参考图

支持两种方式：

1. 点击文件选择上传图片。
2. 鼠标移入图片生成区域，按 `Ctrl + V` 粘贴剪贴板图片。

最多支持：

```text
9 张参考图
```

生成时会把这些图片全部作为多图参考传给 Gemini。

### 17.2 图片生成提示词

在 `图片提示词` 中描述你想生成的画面。

建议写清楚：

- 主体是谁。
- 参考哪张图的人物。
- 参考哪张图的场景。
- 画面风格。
- 构图。
- 光线。
- 是否需要保持角色一致。

示例：

```text
参考图1的人物造型，参考图2的走廊场景，生成一张低多边形3D风格的紧张跑酷画面，角色正在向镜头冲来，背景有轻微运动模糊。
```

### 17.3 生成图片

点击：

```text
生成图片
```

生成结果会显示在图片生成区。

生成图片会保存到：

```text
05_Video/uploads/
```

### 17.4 放大查看生成图

点击生成出的图片，可以打开大图查看。

关闭方式：

- 点击右上角关闭按钮。
- 点击暗色背景。
- 按 `Esc`。

### 17.5 加入视频参考素材

生成图片后，可以点击：

```text
加入参考图
```

图片会被加入当前分镜任务的视频参考素材中。

## 18. 鼠标悬停粘贴规则

页面支持按鼠标悬停区域区分粘贴目标：

| 鼠标位置 | Ctrl + V 结果 |
| --- | --- |
| 左侧视频生成区 | 粘贴到视频参考素材 |
| 右侧图片生成区 | 粘贴到图片生成参考图 |

注意：

- 粘贴只支持图片。
- 视频不能通过剪贴板粘贴，请用文件选择上传。

## 19. 文件保存位置

### 19.1 上传素材

上传的图片、视频、首尾帧、生成图片会保存到：

```text
05_Video/uploads/
```

### 19.2 分镜片段与导出成片

生成片段和导出视频保存到：

```text
05_Video/segments/
05_Video/exports/
```

### 19.3 工作区配置

集数、任务、Prompt、参考素材路径等配置保存到：

```text
05_Video/workspace/episodes.json
```

### 19.4 API Key

密钥保存到：

```text
API_Key/
```

不要提交或外传。

## 20. 常见问题

### 20.1 页面没有新功能

请按：

```text
Ctrl + F5
```

强刷页面。

### 20.2 显示服务版本过旧

关闭旧 GUI 后重新启动：

```bat
start_seedance_gui.bat
```

或者让 Cursor Agent 执行重启脚本。

### 20.3 视频生成失败，提示 API 未发送

优先看任务右侧运行日志。

常见原因：

- Prompt 为空。
- 参考素材缺失。
- API Key 配置错误。
- Seedance API 返回参数错误。

### 20.4 生成视频一直等待

可能是：

- 正在编码大图片。
- 正在上传参考视频。
- Seedance API 队列较慢。
- 网络不稳定。

请查看运行日志中的最新行。

### 20.5 下载按钮没有保存到期望目录

下载目录由浏览器控制。

请检查浏览器设置：

- 默认下载文件夹。
- 是否开启“下载前询问保存位置”。

## 21. 推荐工作流

推荐按以下顺序使用：

1. 新建集数。
2. 添加多个分镜任务。
3. 为每个分镜填写 Prompt。
4. 上传参考图或参考视频。
5. 设置时长和画幅比例。
6. 必要时使用 AI 提示词优化。
7. 点击干跑检查。
8. 点击正式生成。
9. 查看每个分镜结果。
10. 下载单个分镜视频或继续调整。
11. 点击一键拼接本集。
12. 下载拼接成片。

## 22. 使用建议

- 每个分镜尽量只表达一个清晰动作或镜头变化。
- Prompt 中写清楚时间段和镜头语言。
- 参考图越清晰，角色一致性通常越好。
- 参考视频建议短、清晰、动作明确。
- 多张参考图时，Prompt 中最好说明每张图的用途。
- 正式生成前可以先点干跑检查。
- 复杂镜头建议拆成多个短分镜，再用一键拼接合成。
